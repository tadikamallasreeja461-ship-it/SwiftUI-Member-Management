# SwiftUI Member Management

This project is a SwiftUI application built using MVVM architecture.

## Features
- Add Member (Name, Email, Phone, Gender)
- Input validation
- List display
- Detail screen
- Delete member
- State management using ObservableObject
- Navigation using NavigationStack

## Architecture
- MVVM Pattern
- SwiftUI Views
- ObservableObject ViewModel
- Singleton Store for in-memory persistence

## How to Run
1. Open TestProject.xcodeproj
2. Select simulator
3. Run the project
